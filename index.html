Mengenal Semantic Elements Di HTML

•    Artikel

Ketika kita ingin membuat layout untuk menampilkan block artikel-artikel yang ada di website kita, tag article bisa kita gunakan dengan element header dan footer . Karena tag article memiliki type block , artinya element ini memiliki lebar sesuai lenbar browser jika tidak menentukan lebarnya.

•    Aside

Element ini bisa kita gunakan untuk melengkapi layout utama . Misal kita memiliki halaman bllog maka kita bisa menanmbahkan aside di abgian bawah postingan untuk menjelaskan ide . Karena aside memiliki type/sifat block maka lebarnya sesyau page jika kita tidak menentukan lebarnya.
